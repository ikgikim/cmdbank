액셀 지능형.자동채우기 카피 복붙 아래셀에서 ctrl e 

. 파일 이름만 클립보드로 저장  (/a:-d)

dir /a:-d /b |clip

2, 파일 이름만 클립보드로 저장 (하부 폴더 모두 포함 /s)

dir /a:-d /b /s |clip

3. 하부 폴더 이름만 클립보드로 저장 (/ad or /a:d) 

dir /a:d /b |clip
or dir /ad /b |clip

dir /a:d /b /s |clip
(하부 폴더 포함)

4. 폴더명, 파일명 모두 복사하고 싶을때 (폴더 먼저, 파일 나중순 /o:g)

dir /o:g /b |clip

※ 파일로 저장 
dir /a:-d /b >list.txt
dir /a:-d /b /s >list.txt
